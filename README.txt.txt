Bonjour,

Pour lancer le site rendez vous dans le dossier www puis cliquez sur l'index.html.

Ce site a été développé en HTML5, SASS, Angular et Javascript. Vous retrouverez les différents fichiers Sass dans le dossier src/assets/scss et les différents fichiers Javascript dans le dossier src/assets/js
Vous pouvez également lancer l'application sur support mobile en utilisant cordova via les commandes cordova run IOS et cordova run Android

Fonctionnalité:
- Géolocalisation
- Local Storage
- Responsive Design
- Meteo
- Ittinéraire avec Maps

Cordialement.
